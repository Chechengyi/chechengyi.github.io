# 插入排序和希尔排序

# 普通插入排序

function insertSort(arr){
  for ( var i=1; i<arr.length; i++ ) {
    for ( var j=i; j>0; j-- ) {
      if ( arr[j-1] > arr[j] ) {
        // 交换数组中两个元素的操作
        swap(arr, j, j-1)  
      } else {
        break
      }
    }
  }
}

这个是可以改进的，之前的插入排序版本是每遇到前一个元素大于后一个元素就要交换一次两个元素的位置，实际上可以不这么做，改进后的版本为：

function insertSort(arr){
  for ( var i=0; i<arr.length;i++ ) {
    var mid = arr[i]
    for (var j=i; j>0; j--) {
      if ( arr[j-1] > mid ) {
         arr[j] = arr[j-1] 
      } else {
        break
      }
    }
    arr[j] = mid
  }
}

# 希尔排序

希尔排序就是插入排序的改良版本。试想插入排序， 如果是 [1,2,3, 5, 4] 这样的数组，在 j = 3 之前的循环里都是直接 break 了， 因为前面的数组都已经是排好序了的了， 如果下标为 j-1 的元素小于下标为 j 的元素，j-1 之前的元素也不可能比 j 这个位置的元素大，由此可见，在数组近乎有序的情况下，插入排序的效率是很高的。

希尔排序由此诞生，它先确定了一个增量，比如 5 。然后将数组中每隔 5 的长度的元素排好序。 然后增量逐渐递减，递减一次排一次， 在最后一次这个增量等于 1 的时候，其实就是一次插入排序了。 而这个时候的数组比起最初已经有序多了。

function shellSort(arr){
  var gap = Math.floor(arr.length/2)
  for ( gap; gap>0; gap = Math.floor(gap/2) ) {
    for ( var i=gap; i<arr.length; i++ ) {
      var j = i
      var mid = arr[i]
      while (j > 0) {
        if ( arr[j-gap] > mid ) {
           arr[j] = arr[j-gap] 
        } else {
          break
        }
        j -= gap
      }
      arr[j] = mid
    }
  }
}
